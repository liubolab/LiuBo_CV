## 📚 Selected Publications

### DiffuGC: Diffusion Model Can Help Discover Granger Causality from Observational Interventional Time Series  
**ICDM 2025, CCF-B, Oral**  
First Author  
- Proposed a diffusion-based framework for Granger causal discovery under intervention.
- Explored causal intervention without relying on strong prior knowledge.

### LLM-GC: Advancing Granger Causal Discovery from Time Series with Multimodal Language Modeling  
**WSDM 2025, CCF-B**  
First Author  
- Investigated four paradigms for applying large language models to Granger causal discovery.
- Proposed bidirectional retrieval alignment to stimulate LLM-based causal reasoning.
- Achieved state-of-the-art performance on six public datasets.

### From Knowledge to Causality: Self-Supervised Representation Learning for Granger Causal Discovery in Groups of Time Series  
**DASFAA 2025, CCF-B**  
First Author  
- Designed a knowledge-conditioned attention mechanism for group-level time-series representation learning.
- Improved causal discovery by reducing knowledge loss in group representations.

### Spiking Meets Causality: Efficient Granger Causal Discovery With Spiking Neural Networks  
**ICASSP 2026, CCF-B**  
First Author  
- Proposed causality-aware spiking attention for efficient and low-energy causal discovery.

### A General Aircraft Load Model through Two-Phase Prediction Process with Deep Learning Multi-Model Architecture  
**Communications Engineering, Nature Portfolio, 2023**  
Core Technical Contributor  
- Built a two-stage deep learning framework for aircraft load prediction.
- Achieved high-accuracy strain prediction on large-scale real flight records.
